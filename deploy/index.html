    # docker操作

    # 拉取镜像

    docker pull 地址

    # 创建启动容器

    docker run -itd -p 主机端口：映射端口 --name trunkos -v 主机路径:容器路径

    示例: docker run -itd --name hhh -v /home/hhh:/home/hhh -p 6255:22 -p 8080:8080 -p 80:80 镜像ID（镜像ID可通过docker images查看）

    # 查看容器的基本信息

    docker inspect 容器ID

    # 进入容器

    docker exec -it hhh /bin/zsh

    # 切换到对应目录下

    cd /home/

    • 首次操作

    git clone 仓库地址

    • 非首次执行

    git pull

    # 切换路径并下载依赖

    cd /hhh

    cnpm install

    # 打包

    npm run electron:build

    打包完成后 退出容器，进入宿主机的目录
