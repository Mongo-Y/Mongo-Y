# 自定义组件

设计背景: vue项目通过electron打包 。e-charts水球图组件在PC端网页端实现流畅无误，但是在electron应用打包之后阻塞无法显示。所以决定新写一个组件。

从以上几个角度，对组件进行设计，尽可能的考虑到使用时的每一种情况，实现可复用性强的组件。

通过统一的option设置，可以对大小 颜色 标题等元素进行定制化操作。

动态传入value值和ID值，传入ID值的目的是考虑到同一页面可能存在多个水球图，ID相同将无法加载的问题。
